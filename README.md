## 🖥️ What RuleScope Does

RuleScope is a desktop app for reading regulation and policy documents on Windows. It helps you browse long documents, search text fast, and build outlines from headings and numbered sections.

It also recognizes Word auto-numbering, so documents with nested lists and formal structure are easier to read. That makes it useful for policy files, legal drafts, standards, and internal rulebooks.

## 🧩 Troubleshooting

If a document does not look right:

1. Check that the file opens in Word or another reader first.
2. Make sure the document has headings or numbered sections.
3. Try a smaller file to test the app.
4. Close the app and open it again.
5. Re-download the file if the document seems damaged.

If search does not return results, try a shorter phrase or check the spelling.

If the outline looks incomplete, the file may use formatting that is not easy to detect. A document with clear heading styles usually works best.
